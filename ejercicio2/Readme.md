# Ejercicio 2

## Mascota

* En este ejercicio creamos un sistema para simular el cuidado de una mascota virtual 

# Con los siguientes objetivos 

1. Dibuja el diagrama UML de la clase Mascota con todos sus 
atributos y métodos
2. Implementa la clase en Python siguiendo el diagrama UML
3. Crea al menos 2 objetos de tipo Mascota con diferentes características
4. Prueba todos los métodos de la clase y muestra los resultados

* Para esta mascota debemos agregar una validacion en los metodos **alimentar** y **jugar**, con  la condicion de que la felicidad no supere el 100%
