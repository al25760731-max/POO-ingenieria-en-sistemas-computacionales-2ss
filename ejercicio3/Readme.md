# Ejercicio 3 

## Clase platillos 

* En este ejercicio aprendimos a como funciona la **Herencia** de atributos de una clase a otra para ello creamos una clase **padre** y 3 clases **hijas** donde las clases hijas son las que heredan los atributos de la clase padre 

1. **Clase padre** platillo: Esta es la clase de la cual las demas van a hererar las caracteristicas  
2. **Hija 1** Comida: A esta clase se nos pido agregar el atributo categoria y poder sobre escribir el tipo, para que imprima el tipo de comida y su categoria 
3. **Hija 2** Bebida: En esta clase se nos pido el atrubuto de temperatura (frio o caliente), y poder sobre escribir el tipo para que imprima tipo de bebida y su temperatura 
4. **Hija 3** Postre: Para esta clase se nos pido el atrubuto es_sin_gluten (true o false), para que sobreescribe el tipo y que imprima tipo de poste y diga sin gluten si/no
